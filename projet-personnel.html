Projet M2L en Java

Description : Ce projet consiste à développer une application pour la gestion des ressources de la Maison des Ligues de Lorraine (M2L). Il permet de gérer les employés, les réservations et les infrastructures.

Fonctionnalités

  • Gestion des employés avec dates d'arrivée et de départ
  • Gestion des salles et des équipements
  • Système de réservation en ligne
  • Authentification et rôles utilisateurs

Technologies utilisées

✔️ Java (Spring Boot) | ✔️ MySQL | ✔️ Git
